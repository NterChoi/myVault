| 항목   | 사용 예         |
| ------ | --------------- |
| 정수   | 123, -345, 0    |
| 실수   | 123.45, -1234.5 |
| 복소수 | 1 + 2j, -3j     |
| 8진수  | 0o34, 0o25      |
| 16진수 | 0x2A, 0xFF                |

>[!note]  정수형
정수를 뜻하는 자료형

> [!note] 실수형 
> 소수점이 포함된 숫자를  표현하는 자료형

>[!note] 8진수와 16진수
> - 8진수를 만들기 위해서는 0o 또는 0O(숫자 0 + 알파벳 소문자 o 또는 대문자 O)
> - 16진수를 만들기 위해서는 0x 또는 0X숫자 0 + 알파벳 소문자 x 또는 대문자 X)
