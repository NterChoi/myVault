| 항목   | 사용 예         |
| ------ | --------------- |
| 정수   | 123, -345, 0    |
| 실수   | 123.45, -1234.5 |
| 복소수 | 1 + 2j, -3j     |
| 8진수  | 0o34, 0o25      |
| 16진수 | 0x2A, 0xFF                |


>[!note] 8진수와 16진수
> - 8진수를 만들기 위해서는 0o 또는 0O(숫자 0 + 알파벳 소문자 o 또는 대문자 O)
> - 16진수를 만들기 위해서는 0x 또는 0X숫자 0 + 알파벳 소문자 x 또는 대문자 X)

>[!note] 복소수
>- 복소수(Complex number)를 배울 때 i를 사용했지만, 파이썬은 i 대신 j를 사용
>- 소문자 j 또는 대문자 J를 사용
>- 복소수 관련 내장 함수를 사용하기 위해 변수 이름 뒤에 .을 붙인 후 함수 이름을 사용함
> 	 - 복소수 .real은 복소수의 실수 부분을 반환
> 	 - 복소수.imag은 복소수의 허수 부분을 반환함

> [!caution] 파이썬 산술 연산자
> ** = x의 y 제곱을 나타내는 연산자
> ```python
> a = 3
> b = 4
> a ** b
> 81 
> ```
> // = 나눗셈 후 몫을 반환하는 연산자
> ```python
> 7 / 4 
> 1.75
> 
> 7 // 4
> 1
> ```
