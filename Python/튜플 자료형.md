> 튜플은 몇 가지 점을 제외하고는 리스트와 거의 동일하며 다른 점은 다음과 같음
> - 리스트는 대괄호로 둘러싸지만, 튜플은 소괄호로 둘러쌈
> - 리스트는 그 값의 생성, 삭제, 수정이 가능하지만 튜플은 그 값을 바꿀 수 없음


```python
>>> t1 = ()
>>> t3 = (1,2,3)
>>> t4 = 1,2,3 # 괄호를 생략해도 무방함
>>> t5 = ('a', 'b', ('ab', 'cd'))
```
