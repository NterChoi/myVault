> 리스트를 만들 때는 대괄호로 감싸 주고 각 요소값들은 쉼표로 구분한다.
> 리스트명 = [요소1, 요소2, 요소3, ···]

```python
a = []                              # 비어있는 리스트를 가질 수 있다.
b = [1,2,3]                         
c = ['Life', 'is', 'too', 'short']
d = [1,2,'Life', 'is']
e = [1,2,['Life','is']]              # 리스트 자체를 요소값으로 가질 수 있다.
```

### 리스트의 인덱싱


```python
>>>a = [1,2,3] # a == 리스트
>>>a[0] # a[0] == 첫번째 요소 == 숫자형 1
1

>>>a[0] + a[2] # 리스트 첫 번째 요소 + 세 번째 요소 == 그냥 연산
4

>>>a[-1] # a[-1] == 해당 리스트의 마지막 요소
3

>>> a = [1,2,3,['a', 'b', 'c']]

>>> a[0]
1
>>> a[-1]
['a', 'b', 'c']

>>> a[-1][0] # 리스트에 a에 포함된 ['a','b','c']라는 리스트 값 인덱싱하는 방법
'a'

>>> a[-1][1]
'b'

>>> a[-1][2]
'c'
```

### 리스트의 슬라이싱

```python
>>> a = [1,2,3,4,5]
>>> a[0:2] # a[0] ~ a[1]의 리스트 의미
[1,2]

>>> b = a[:2] # 리스트 a의 첫번째 요소부터 a[1]까지의 리스트를 의미
>>> c = [2:]  # 리스트 a의 a[2]부터 마지막 요소까지의 리스트를 의미 즉 슬라이싱에서 [a:b]로 할때 b는 포함X
>>> b
[1,2]
>>> c
[3,4,5]
```

> 리스트 연산자
> + 기호를 이용해 더할 수 있고, * 기호를 이용해서 반복할 수 있음

```python
>>> a = [1,2,3]
>>> b = [4,5,6]
>>> a + b # 리스트 사이에서 + 기호는 2개의 리스트를 합치는 기능
[1,2,3,4,5,6]

>>> a * 3 # * == 반복
[1,2,3,1,2,3,1,2,3,] # [1,2,3]이라는 리스트가 세 번 반복되어 새로운 리스트를 만듬
```

> 리스트의 수정과 삭제

```python
>>> a = [1,2,3]
>>> a[2] = 4 # 리스트에서 하나의 값 수정
>>> a
[1,2,4]

>>> a = [1,'a', 'b', 'c', 4]
>>> a[1:3] = [] # a[1:3]을 [ ]으로 바꿔 주었기 때문에 ['a', 'b']가 삭제됨
>>> a
[1, 'c', 4]

>>> del a[1] # del a[x]는 x번째 요소값을 삭제하고 del[x:y]는 x번째부터 y-1번째 요소까지의 값을 삭제함
>>> a
[1, 4]
```

> 리스트 관련 함수들

>append(x) == 리스트의 맨 마지막에 x를 추가시키는 함수

```python
>>> a = [1,2,3]
>>> a.append(4) # append == 추가 
>>> a
[1,2,3,4]

>>> a.append([5,6]) # 리스트에는 어떤 자료형도 추가가능
>>> a
[1,2,3,4,[5,6]] 
```

> sort( ) == 리스트의 요소를 오름차순으로 정렬함
> reverse( ) == 리스트의 요소를 역순으로 정렬함

```python
>>> a = [1,4,3,2]
>>> a.sort()
>>> a
[1,2,3,4]

>>> a = ['a', 'c', 'b']
>>> a.sort()
>>> a
['a', 'b', 'c']

>>> a = ['a', 'c', 'b']
>>> a.reverse()
>>> a
['b', 'c', 'a']
```

> [!question]+ sort( )를 이용하여 내림차순 정렬하기
> 리스트.sort( ) 함수에는 reverse 옵션(매개변수)이 존재한다.
> 디폴트는 리스트.sort(revers = False)인게 디폴트여서 리스트가 오름차순으로 정렬된다.
> 리스트.sort(reverse = True)로 변경하면 리스트가 내림차순으로 정렬된다.

```python
>>> a = [3, 2, 8, 4, 1, 10, 99, 5]
>>> b = [3, 2, 8, 4, 1, 10, 99, 5]
>>> c = [3, 2, 8, 4, 1, 10, 99, 5]

# 기본값 (오름차순)
>>> a.sort()
>>> a
[1,2,3,4,5,8,10,99]

# 오름차순
>>> b.sort(reverse=False)
>>> b
[1,2,3,4,5,8,10,99]

# 내림차순
>>> c.sort(reverse=True)
>>> c
[99, 10, 8, 5, 4, 3, 2, 1]
```
