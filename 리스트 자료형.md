> 리스트를 만들 때는 대괄호로 감싸 주고 각 요소값들은 쉼표로 구분한다.
> 리스트명 = [요소1, 요소2, 요소3, ···]

```python
a = []                              # 비어있는 리스트를 가질 수 있다.
b = [1,2,3]                         
c = ['Life', 'is', 'too', 'short']
d = [1,2,'Life', 'is']
e = [1,2,['Life','is']]              # 리스트 자체를 요소값으로 가질 수 있다.
```

### 리스트의 인덱싱


```python
>>>a = [1,2,3] # a == 리스트
>>>a[0] # a[0] == 첫번째 요소 == 숫자형 1
1

>>>a[0] + a[2] # 리스트 첫 번째 요소 + 세 번째 요소 == 그냥 연산
4

>>>a[-1] # a[-1] == 해당 리스트의 마지막 요소
3

>>> a = [1,2,3,['a', 'b', 'c']]

>>> a[0]
1
>>> a[-1]
['a', 'b', 'c']

>>> a[-1][0] # 리스트에 a에 포함된 ['a','b','c']라는 리스트 값 인덱싱하는 방법
'a'

>>> a[-1][1]
'b'

>>> a[-1][2]
'c'
```

### 리스트의 슬라이싱

```python
>>> a = [1,2,3,4,5]
>>> a[0:2] # a[0] ~ a[1]의 리스트 의미
[1,2]

>>> b = a[:2] # 리스트 a의 첫번째 요소부터 a[1]까지의 리스트를 의미
>>> c = [2:]  # 리스트 a의 a[2]부터 마지막 요소까지의 리스트를 의미 즉 슬라이싱에서 [a:b]로 할때 b는 포함X
>>> b
[1,2]
>>> c
[3,4,5]
```

> 리스트 연산자
> + 기호를 이용해 더할 수 있고, * 기호를 이용해서 반복할 수 있음

```python
>>> a = [1,2,3]
>>> b = [4,5,6]
>>> a + b # 리스트 사이에서 + 기호는 2개의 리스트를 합치는 기능
[1,2,3,4,5,6]

>>> a * 3 # * == 반복
[1,2,3,1,2,3,1,2,3,] # [1,2,3]이라는 리스트가 세 번 반복되어 새로운 리스트를 만듬
```
