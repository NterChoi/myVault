> 리스트를 만들 때는 대괄호로 감싸 주고 각 요소값들은 쉼표로 구분한다.
> 리스트명 = [요소1, 요소2, 요소3, ···]

```python
a = []                              # 비어있는 리스트를 가질 수 있다.
b = [1,2,3]                         
c = ['Life', 'is', 'too', 'short']
d = [1,2,'Life', 'is']
e = [1,2,['Life','is']]              # 리스트 자체를 요소값으로 가질 수 있다.
```

### 리스트의 인덱싱


```python
>>>a = [1,2,3]
>>>a[0]
1

>>>a[0] + a[2]
4

>>>a[-1]
3

>>> a = [1,2,3,['a', 'b', 'c']]

>>> a[0]
1
>>> a[-1]
['a', 'b', 'c']

>>> a[-1][0]
'a'

a[-1][1]
'b'

```
