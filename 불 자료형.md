불 (bool) 자료형이란 참(True)과 거짓(False)을 나타내는 자료형이다.
불 자료형은 다음 2가지 값만을 가진다.

- True : 참을 의미
- False : 거짓을 의미

>True나 False 는 파이썬의 예약어로, true, false와 같이 작성하면 안되고 첫 문자를 항상 대문자로 작성해야 한다.

### 불 자료형은 어떻게 사용할까?

```python
>>> a = True # 따옴표로 감싸지 않은 문자열을 변수로 지정해서 오류가 발생할 것 같지만 잘 실행된다.
>>> b = False 

>>> type(a) # type 함수를 변수 a와 b로 사용하면 두 변수의 자료형이 bool로 지정된것을 확인 가능.
<class 'bool'>
>>> type(b)
<class 'bool'>

```

> type(x)는 x의 자료형을 확인하는 파이썬의 내장 함수이다.

불 자료형은 조건문의 리턴값으로도 사용된다.

```python
>>> 1 == 1 # 1 == 1은 '1과 1은 같은가?'를 묻는 조건문이다. 
True # 이러한 조건문의 결과로는 True 또는 False에 해당하는 불 자료형을 리턴한다.

>>> 2 > 1 # 조건문이 참이면 True
True

>>> 2 < 1 # 조건문이 거짓이면 False
False

```

### 자료형의 참과 거짓
-----------------------------------
| 값        | 거짓 |
| --------- | ---- |
| "python"  | 참   |
| " "       | 거짓 |
| [1,2,3]   | 참   |
| []        | 거짓 |
| (1,2,3)   | 참   |
| ( )       | 거짓 |
| {'a' : 1} | 참   |
| { }       | 거짓 |
| 1         | 참   |
| 0         | 거짓 |
| None      | 거짓 |

문자열, 리스트, 튜플, 딕셔너리 등의 값이 비어 있으면("", )
