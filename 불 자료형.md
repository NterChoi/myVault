불 (bool) 자료형이란 참(True)과 거짓(False)을 나타내는 자료형이다.
불 자료형은 다음 2가지 값만을 가진다.

- True : 참을 의미
- False : 거짓을 의미

>True나 False 는 파이썬의 예약어로, true, false와 같이 작성하면 안되고 첫 문자를 항상 대문자로 작성해야 한다.

### 불 자료형은 어떻게 사용할까?
--------------------------------------------


```python
>>> a = True # 따옴표로 감싸지 않은 문자열을 변수로 지정해서 오류가 발생할 것 같지만 잘 실행된다.
>>> b = False 

>>> type(a) # type 함수를 변수 a와 b로 사용하면 두 변수의 자료형이 bool로 지정된것을 확인 가능.
<class 'bool'>
>>> type(b)
<class 'bool'>

```

> type(x)는 x의 자료형을 확인하는 파이썬의 내장 함수이다.

불 자료형은 조건문의 리턴값으로도 사용된다.

```python
>>> 1 == 1 # 1 == 1은 '1과 1은 같은가?'를 묻는 조건문이다. 
True # 이러한 조건문의 결과로는 True 또는 False에 해당하는 불 자료형을 리턴한다.

>>> 2 > 1 # 조건문이 참이면 True
True

>>> 2 < 1 # 조건문이 거짓이면 False
False

```

### 자료형의 참과 거짓
-----------------------------------
| 값        | 거짓 |
| --------- | ---- |
| "python"  | 참   |
| " "       | 거짓 |
| [1,2,3]   | 참   |
| []        | 거짓 |
| (1,2,3)   | 참   |
| ( )       | 거짓 |
| {'a' : 1} | 참   |
| { }       | 거짓 |
| 1         | 참   |
| 0         | 거짓 |
| None      | 거짓 |

문자열, 리스트, 튜플, 딕셔너리 등의 값이 비어 있으면("", [], (), {}) 거짓이 되고 비어 있지 않으면 참이 된다.
숫자에서는 그 값이 0일때 거짓이 된다. 
None은 그저 거짓을 뜻한다는 것만 알아두자 뒷부분에서 배운다

다음 코드를 보고 자료형의 참과 거짓이 프로그램에서 어떻게 쓰이는지 알아보자.

```python
>>> a = [1, 2, 3, 4] # a = [1,2,3,4]라는 리스트를 만듬
>>> while a: # 조건문이 참인 동안 조건문 안에 있는 문장을 반복해서 수행
...     print(a.pop())
...
4
3
2
1

```

```python
while 조건문:
    수행할_문장

```

위 코드를 보면 a가 참인 경우, a.pop()를 계속 실행하여 출력하라는 의미이다.
a.pop() 함수는 리스트 a의 마지막 요소를 끄집어 내는 함수 이므로 리스트 안에 요소가 존재하는 한 (a가 참인 동안) 마지막 요소를 계속 끄집어 낼 것이다.
결국 더 이상 끄집어 낼 것이 없으면 a가 빈 리스트( [] )가 되어 거짓이 된다.
따라서 while 문에서 조건문이 거짓이 되므로 while 문을 빠져 나가게 된다.

다른 예제 코드를 살펴보자

```python
>>> if []:
...     print("참")
... else:
...     print("거짓")
...
거짓

```

[]는 앞의 표에서 볼 수 있듯이 비어 있는 리스트이므로 거짓이다.
따라서 "거짓"이라는 문자열이 출력된다.

### 불연산
-------------------------

bool 함수를 사용하면 자료형의 참과 거짓을 보다 정확하게 식별할 수 있다.

```python
>>> bool('python') # 'python'이라는 문자열이 비어 있지 않으므로 bool 연산의 결과로 True를 리턴
True

>>> bool([1, 2, 3])
True
>>> bool([])
False
>>> bool(0)
False
>>> bool(3)
True

```
