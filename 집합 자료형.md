### 집합 자료형은 어떻게 만들까
-------------------------------
집합 자료형은 다음과 같이 set 키워드를 사용해 만든다.

```python
>>> s1 = set([1, 2, 3]) # set()의 괄호 안에 리스트를 입력하여 만들거나
>>> s1
{1, 2, 3}

>>> s2 = set("Hello") # 문자열을 입력하여 만들 수 있다.
>>> s2
{'e', 'H', 'l', 'o'}
# 비어 있는 집합 자료형은 s = set()으로 만들 수 있다. 
```

### 집합 자료형의 특징
----------------------------
> [!info]+ 
> - 중복을 허용하지 않는다.
> - 순서가 없다(Unordered).
> 	- 순서가 없기(unorderd) 때문에 인덱싱을 통해 요소값을 얻을 수 없다.

set 자료형에 저장된 값을 인덱싱으로 접근하려면 다음과 같이 리스트나 튜플로 변환 후 접근해야 한다.

```python
>>> s1 = set([1, 2, 3])
>>> l1 = list(s1)
>>> l1
[1, 2, 3]
>>> l1[0]
1
>>> t1 = tuple(s1)
>>> t1
(1, 2, 3)
>>> t1[0]
1
```

### 교집합, 합집합, 차집합 구하기
---------------------------------------

#### 교집합 구하기

앰퍼샌드(&) 기호를 이용하거나, intersection 함수를 사용하면 구할 수  있다.

```python
>>> s1 = set([1, 2, 3, 4, 5, 6])
>>> s2 = set([4, 5, 6, 7, 8, 9])

>>> s1 & s2
{4, 5, 6}

>>> s1.intersection(s2) # s2.intersection(s1)을 사용해도 결과는 동일
{4, 5, 6}

```

#### 합집합 구하기

바(|) 기호를 이용하거나 union 함수를 사용하면 구할 수 있다.

```python
>>> s1 = set([1, 2, 3, 4, 5, 6])
>>> s2 = set([4, 5, 6, 7, 8, 9])

>>> s1 | s2
{1, 2, 3, 4, 5, 6, 7, 8, 9}

>>> s1.union(s2) # s2.union(s1)을 사용해도 결과는 동일
{1, 2, 3, 4, 5, 6, 7, 8, 9}

```

#### 차집합 구하기

빼기(-) 기호를 이용하거나 difference 함수를 사용하면 구할 수 있다.

```python
>>> s1 = set([1, 2, 3, 4, 5, 6])
>>> s2 = set([4, 5, 6, 7, 8, 9])

>>> s1 - s2
{1, 2, 3}
>>> s2 - s1
{8, 9, 7}

>>> s1.difference(s2)
{1, 2, 3}
>>> s2.difference(s1)
{8, 9, 7}

```

### 집합 자료형 관련 함수
-----------------------
#### 값 1개 추가하기 - add()

이미 만들어진 set 자료형에 1개의 값을 추가할 수 있다.

```python
>>> s1 = set([1, 2, 3])
>>> s1.add(4)
>>> s1
{1, 2, 3, 4}

```

#### 값 여러개 추가하기 - update()

여러개의 값을 한번에 추가(update)할 수 있다.

```python
>>> s1 = set([1, 2, 3])
>>> s1.update([4, 5, 6])
>>> s1
{1, 2, 3, 4, 5, 6}

```

#### 특정 값 제거하기 - remove()

특정 값을 제거하고 싶을 때 사용할 수 있다.

```python
>>> s1 = set([1, 2, 3])
>>> s1.remove(2)
>>> s1
{1, 3}

```

> [!faq]+ remove()로 여러 요소값 삭제하기
> remove() 는 한 번에 하나의 요소만 삭제할 수 있다.
> 따라서 여러 요소를 한 번에 삭제하려면 반복문을 사용해야한다.

```python
# 집합을 생성합니다.
my_set = {1, 2, 3, 4, 5}

# 삭제할 요소를 리스트로 만듭니다.
elements_to_remove = [2, 4]

# 반복문을 사용하여 요소를 삭제합니다.
for element in elements_to_remove:
    if element in my_set:
        my_set.remove(element)

# 결과를 출력합니다.
print(my_set)  # {1, 3, 5}

```
